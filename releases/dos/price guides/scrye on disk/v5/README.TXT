SCRYE ON DISK VOL #5
SCRYE ON DISK contains the latest prices for collectible card games as published in SCRYE magazine Issue #5 as of January 15, 1995

There are 2 file types available on SCRYE ON DISK:
	1. *.txt - tab delimited text format for use with Spreadsheet, Database, Word Processor or Game Card Collector programs.
	2. *.sod - format for use with Gamecard Wizard

In the *.txt files, the column after the Median Price contains the up and down arrows.  These arrows are in TrueType WingDing Font.  It may be necessary to change this column to this font to see the arrows or to perform a Search and Replace in a word processing program.  In normal font "é" represents up and "ê" represents down.
